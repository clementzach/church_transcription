How to Use

Are you a listener?

  • This page is for the broadcaster (the person running the microphone). If you want to read captions or hear translated audio, go to the Listener page ↗ instead.

Starting a session

  • Click Start and allow microphone access when prompted. A session ID will appear in the top bar.
  • Share the session ID with listeners so they can follow along on their devices at the Listener page.
  • Click Stop to end the session. Sessions automatically expire after 2 hours.

Spoken language

  • AUTO — Gladia detects the spoken language automatically. Use this when the speaker may switch languages.
  • EN / ES / HT / PT / ZH / FR — Pin to a specific spoken language for faster, more accurate transcription. All other languages still receive translated captions.
  • Changing this setting while live silently reconnects the session.

Caption panels

  • The grid shows live captions in up to six languages simultaneously: English, Spanish, Haitian Creole, Portuguese, Mandarin, and French.
  • The most recent line is shown largest; older lines fade and shrink above it.
  • Italic text in a panel is a live partial (in-progress utterance); it becomes solid once the sentence is finalised.

Show panels buttons

  • Toggle individual language panels on or off. At least one panel must remain visible.
  • Hidden panels are dimmed with a transparent background; visible panels are solid-coloured.

Other controls

  • ⛶ — Enter fullscreen. The header is hidden in fullscreen for a clean caption display.
  • Hide controls (bottom bar) — Hides the header without going fullscreen. Click Show controls to bring it back.
  • Listener ↗ — Opens the Listener page in a new tab so you can preview what remote listeners see.
